## Xcode Setup

1. Öffne `Assets.xcassets` in Xcode
2. Wähle `AppIcon`
3. Ziehe die passende PNG-Datei in jeden Slot (Größen sind im Dateinamen)
4. Für App Store Submission: 1024×1024 verwenden

## Farbpalette

- Bordeaux Gradient: `#7A2A2E` → `#5C1A1E` → `#3A0E12`
- Beeren (Gold): `#F2D88B` → `#D9B574` → `#7E5E2A`
- Blatt / Stiel (Creme): `#F4ECDD`
